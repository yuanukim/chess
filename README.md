# chess
##### chess game implemented in C++14 with a simple AI. AI uses min-max algorithm and alpha-beta pruning, with multi-thread speed up. type "help" when you meets anything unknown.
##### color supports windows, linux, freebsd.
##### In year 2021, I wrote the first version of this program as an exam for my C++ learning. how time flies!
